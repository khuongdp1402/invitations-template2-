# VOLUME 2: UX Flow & Storyboard
*(Máy trạng thái Trục Y - Biến Kịch bản thành Toán học)*

## 1. MỤC TIÊU (OBJECTIVE)
Biến toàn bộ trải nghiệm kể chuyện (Storytelling) thành các điểm neo (Triggers) chính xác tuyệt đối dựa trên tỷ lệ phần trăm cuộn trang (Scroll Percentage). Tài liệu này yêu cầu AI cấu hình GSAP ScrollTrigger.

## 2. QUY TẮC MÁY TRẠNG THÁI (STATE MACHINE RULES)
- **MUST** sử dụng ScrollTrigger để kích hoạt animation.
- **MUST** liên kết trạng thái màu nền (Color of Time) với các mốc Trigger này.
- **NEVER** dùng `IntersectionObserver` thuần. GSAP ScrollTrigger là tiêu chuẩn duy nhất.

## 4. TECHNICAL CONSTRAINTS CHO SCROLLTRIGGER
```javascript
// MUST follow this configuration pattern for sections:
ScrollTrigger.create({
  trigger: sectionRef.current,
  start: "top center", // Khi đỉnh section tới giữa màn hình
  end: "bottom center",
  scrub: 1, // Momentum scrubbing
  onEnter: () => gsap.to(bgRef.current, { backgroundColor: themeColor, duration: 1.5, ease: "power2.out" }),
  onEnterBack: () => gsap.to(bgRef.current, { backgroundColor: themeColor, duration: 1.5, ease: "power2.out" })
});
```
